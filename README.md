# Mario Kart World Map Editor

このリポジトリは、マリオカートのワールドマップを編集・保存できるWebアプリのソースコードです。

## 主な機能
- マップの編集・保存・バックアップ
- GitHub Pagesによる公開

## 開発・実行方法

1. 依存パッケージのインストール
   ```sh
   npm install
   ```
2. 開発サーバーの起動
   ```sh
   npm run dev
   ```
3. ビルド
   ```sh
   npm run build
   ```

## デプロイ
GitHub Actionsで自動的にGitHub Pagesへデプロイされます。
